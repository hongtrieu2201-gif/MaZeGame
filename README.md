# Maze Game - Java Swing

> Mini game mê cung 2D được xây dựng bằng Java Swing để luyện tập OOP, GUI programming, collision detection, animation và tư duy phát triển game cơ bản.

---

## Giới thiệu dự án

**Tên dự án:** Maze Game - Java Swing

**Mục tiêu:**  
Xây dựng một mini game mê cung 2D bằng Java Swing để thực hành lập trình hướng đối tượng, xử lý giao diện, va chạm và animation.

---

## Tính năng chính

- Start Screen đẹp
- Điều khiển bằng `WASD` và phím mũi tên
- Player dùng ảnh PNG
- Enemy dùng ảnh PNG
- Enemy đi tuần
- Coin system
- Lives system
- Random maze có kiểm tra đường đi hợp lệ
- Custom result dialog

---

## Công nghệ sử dụng

- **Java**
- **Java Swing**
- **OOP**
- **Graphics2D**
- **Timer / Animation**

---

## Gameplay

- Người chơi bắt đầu từ ô **Start**
- Di chuyển đến ô **Exit**
- Né quái đi tuần
- Nhặt coin để tăng sao
- Nếu chạm quái sẽ mất 1 mạng và quay về ô Start
